# countmein.io

## Dependencies

1. node.js (nodejs.org)
2. gulp (`[sudo] npm install -g gulp`)

## Setup

```bash
make
```

or

```bash
npm install
gulp
```

## Build 

```bash
gulp
```

### Release

```bash
make release
```

## Run the server and watch for changes

```bash
gulp run
```

## Cleanup

```bash
gulp clean
```

or

```bash
make clean
```
